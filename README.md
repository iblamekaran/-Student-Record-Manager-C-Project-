# Student Record Manager (C Project)

A simple CLI-based student record manager built in C.

## Features
- Uses arrays to store marks
- Uses pointers for calculations
- Implements loops and conditionals
- Calculates average and highest marks
- Assigns grades automatically

## Concepts Used
- Pointers
- Arrays
- Functions
- Loops
- Conditional statements

## How to Run

Compile:
gcc main.c -o student

Run:
./student
